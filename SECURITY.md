# Security Policy

## Supported Versions

Security fixes are handled for the latest code on the `main` branch.

## Reporting a Vulnerability

Please do not open a public issue for security problems.

Use GitHub Security Advisories instead:

https://github.com/HanielCota/Essentialist/security/advisories/new

Include:

- A clear description of the issue.
- Steps to reproduce it.
- The affected version or commit.
- Any relevant logs, configuration, or proof of concept.

Reports will be reviewed as soon as possible.
